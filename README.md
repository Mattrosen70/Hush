# Hush - Meditation & Mindfulness PWA

A progressive web app designed to guide users through personalized meditation experiences with customizable background sounds, guided scripts, voice options, and visual backgrounds.

## Features

- **Background Sounds**: Choose from a variety of ambient sounds (rainfall, crackling fire, forest ambience, ocean waves, etc.)
- **Guided Meditation Scripts**: Multiple meditation programs for different needs (anxiety relief, calming, focus, sleep, etc.)
- **Voice Customization**: Select from different voice options (male, female, various accents) to guide your meditation
- **Duration Control**: Customize meditation session length to fit your schedule
- **Visual Backgrounds**: Choose calming visual backdrops (waterfall, ocean, forest, mountains, etc.)
- **Offline Support**: Works offline with cached assets
- **Progressive Web App**: Install as a native app on your device

## Tech Stack

- Frontend: React/TypeScript
- State Management: Zustand
- Audio: Web Audio API
- Service Worker: Workbox for PWA capabilities
- Styling: Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies: npm install
3. Start development server: npm run dev
4. Build for production: npm run build
